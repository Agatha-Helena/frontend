# 💻 BookWorm

Este é um projeto desenvolvido para a avaliação somativa de Front-End, focado em criar interatividade para um site de clube de leitura de uma livraria.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
* JavaScript (Manipulação de DOM e Eventos)

## 📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:

/somativa-frontend-v3-agatha-helena-silva-monteiro
├── index.html
└── src/
    ├── styles/ style.css (Estilos da página)
    ├── scripts/ script.js (Lógica de interação)
    └── images/ livro.png (Ativos visuais)

## 🚀 Funcionalidades Implementadas
* **Interatividade Visual:** Há elementos que reagem ao passar o mouse em cima, como o caso do card "Clube do Livro" que, ao passar o mouse em cima, ajusta as cores para um modo de leitura mais confortável.
* **Saudação Dinâmica:** Mensagem de boas-vindas são exibidas com base no horário local do usuário do site, sendo elas "Boa leitura matinal!" para antes de 12:00 horas, "Aproveite a tarde para ler!" de 12:00 à 18:00 horas e "Uma boa história antes de dormir?" para depois de 18:00 horas.
* **Cálculo em Tempo Real:** Há também um formulário que processa e exibe resultados assim que o usuário as digita, utilizado no cálculo da compra de livros promocionais que custam R$42,00. Portanto, o cálculo já é feito automaticamente a cada livro adicionado, multiplicando pelo preço do livro.
* **Injeção de Conteúdo:** Para o site, utilizamos a manipulação de DOM, adicionando novos itens na tela e mudando, como utilizado para a mudança de tema no card do começo.
