# 💻 GalaxyTours

Este é um projeto desenvolvido para a avaliação formativa de Front-End, focado em criar media-queries para um site de astronomia.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica)
* CSS3 (Estilização, Feedback Visual e Media Queries)

## 📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado:

/formativa3-mq-agatha-monteiro
├── index.html
├──readme.md
└── src/
    ├── styles/ style.css (Estilos da página)
    ├── scripts/ script.js
    └── images/ foguete.png (Ativos visuais)

## 🚀 Implementações no CSS
* **Interatividade Visual:** Foram utilizados os conceitos de Media Query para adaptar o site ao dispositivo do usuário, fazendo com que a tela corresponda devidamente ao aparelho.
